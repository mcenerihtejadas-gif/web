¿Qué es HTML?

HTML es el lenguaje que permite crear la estructura básica de una página web. Con él organizamos textos, imágenes, enlaces y otros elementos visibles. Funciona mediante etiquetas que indican
al navegador cómo mostrar cada parte. Es el primer lenguaje que debe aprender alguien que inicia en desarrollo web. No es un lenguaje de programación,
sino de marcado para estructurar contenido. Gracias a HTML, las páginas web pueden existir y ser interpretadas por navegadores.

Historia HTML

HTML fue creado en 1989 por Tim Berners-Lee para compartir información científica. Su primera versión tenía pocas etiquetas y era muy simple. Con el tiempo evolucionó hasta
convertirse en un estándar mundial. Organismos como el W3C
definieron reglas y mejoras para garantizar compatibilidad.
Con HTML5 llegaron videos, audios y nuevas etiquetas semánticas. Hoy es la base de todo sitio web moderno.

Editores HTML

Un editor HTML es un programa usado para
escribir y modificar código web. Existen editores simples como el Bloc de Notas y otros avanzados
como VS Code. También hay editores visuales WYSIWYG que muestran
el resultado mientras editas. Cada editor ofrece herramientas para facilitar la escritura del código. Muchos incluyen autocompletado, resaltado y vista previa. Escoger un buen editor ayuda a trabajar más rápido y con menos errores.

Tu primer documento HTML

Un documento HTML debe comenzar con la declaración DOCTYPE que
define su versión. Luego incluye la etiqueta <html> que encierra
todo el contenido. Dentro del <head> se coloca el título y la configuración del documento. En el <body> va todo lo visible, como textos, imágenes y enlaces. Es común iniciar con un título y un párrafo de bienvenida. Este archivo se guarda con extensión .html y se abre en un navegador.

¿Qué son Etiquetas en HTML?

Las etiquetas HTML son instrucciones que indican cómo
mostrar el contenido. Se escriben entre símbolos < > y suelen tener apertura y cierre. Cada etiqueta tiene una función específica dentro del documento. Algunas sirven para texto, otras para imágenes
o para dividir secciones. La correcta estructura de etiquetas garantiza páginas organizadas. Sin etiquetas, un navegador no podría interpretar el contenido de un sitio.

¿Qué son Atributos en HTML?

Los atributos añaden información extra a una etiqueta HTML. Se escriben en la etiqueta de apertura usando la forma nombre="valor". Permiten definir características como tamaño, estilo o comportamiento. Ejemplos comunes son src, id, class, href y placeholder. Cada atributo cumple una función según el tipo de etiqueta donde se use. Sin atributos, muchos elementos no tendrían información esencial para mostrarse.

Reglas fundamentales en HTML

HTML requiere una estructura básica para funcionar correctamente. Las etiquetas deben abrirse y cerrarse en orden para evitar errores. Algunas etiquetas como <img> o <br> no necesitan cierre. Es importante mantener buena indentación para leer el código fácilmente. Los atributos siempre van dentro de la etiqueta de apertura. Seguir estas reglas garantiza páginas más claras y compatibles.

Etiquetas para hacer un documento HTML

Un documento HTML inicia con <!DOCTYPE html> para indicar su versión. La etiqueta <html> es la raíz que contiene todo el contenido. En <head> se coloca el título y la información interna del documento. La etiqueta <meta> permite configurar caracteres especiales como la ñ y tildes. Dentro del <body> va todo el contenido visible de la página. Este conjunto de etiquetas forma la base de cualquier sitio web.

1- ¿Qué es CSS?

CSS significa "Cascading Style Sheets" y se utiliza para definir la apariencia visual de las páginas web. Ayuda a controlar colores, tamaños, posiciones y estilos de los elementos HTML. Sin CSS, las páginas web serían simples documentos sin diseño atractivo. Permite separar la estructura del contenido para que todo esté más organizado. Ayuda a crear diseños modernos y adaptados a diferentes dispositivos. Es una herramienta fundamental para desarrollar sitios web profesionales.

2- ¿Cómo utilizar CSS con HTML?

CSS puede integrarse en un documento HTML de varias formas según la necesidad del proyecto. Una manera es usando estilos directamente en los elementos con el atributo "style". Otra forma es colocar reglas dentro de la cabecera usando la etiqueta <style>. También se puede enlazar un archivo externo que contenga todas las reglas CSS. HTML organiza el contenido mientras CSS se encarga del diseño visual. Al combinar ambos se obtienen páginas completas y bien estructuradas.

3- CSS en línea

El CSS en línea se aplica directamente a un elemento utilizando el atributo "style". Es útil para cambios rápidos y específicos en elementos individuales. No es recomendable para proyectos grandes por su dificultad de mantenimiento. Cada cambio debe realizarse uno por uno y eso hace el código menos ordenado. Aun así es una solución práctica para modificaciones rápidas o pruebas. Debe usarse de manera limitada para evitar desorganización del documento.

4- Interno

El CSS interno se escribe dentro de la cabecera del documento usando <style>. Permite aplicar estilos a toda la página desde un único bloque de reglas. Es práctico para archivos pequeños o proyectos simples que no requieren mucho estilo. Mantiene el contenido HTML separado de los estilos pero dentro del mismo archivo. A medida que crece el proyecto puede volverse difícil de mantener. Es ideal para aprender o para trabajos que no necesitan demasiadas páginas.

5- Externo

El CSS externo utiliza un archivo independiente con extensión .css para almacenar estilos. Este archivo se enlaza al documento HTML mediante <link> en la cabecera. Es la forma más ordenada y profesional de trabajar con estilos web. Permite que varias páginas utilicen el mismo archivo CSS para ahorrar tiempo. Facilita el mantenimiento porque un solo cambio afecta a todo el sitio. Es el método preferido en proyectos grandes y bien estructurados.

Acerca de

El propósito de esta página es ofrecer información clara y organizada sobre la creación de páginas web,
utilizando diferentes recursos y comprendiendo el uso del lenguaje de marcado HTML5.

También presenta un recorrido por las etiquetas más importantes del HTML, junto con una breve introducción al funcionamiento básico de CSS y JavaScript, dos herramientas esenciales para complementar el desarrollo web.
